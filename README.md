# POST-IT
RESTful API created with Nodejs, render.com and MongoDB

## Description
This project is a simple social media app that allows you post anything on a single post-it. 


### Technologies
Project is created with:
* Nodejs 18.14.2
* ExpressJS 4.18.2
* Mongoose 7.0.1
